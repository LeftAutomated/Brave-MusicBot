# Commands

Prefix `/`

## The `help` command
__No arguments__
> /help

## The `play` command
`query` words
> /play query: Just The Two of Us

`query` link
> /play query: https://www.youtube.com/watch?v=PJ0u5c9EF1E

## The `resume` command
__No arguments__
> /resume

## The `pause` command
__No arguments__
> /pause

## The `stop` command
__No arguments__
> /stop

## The `skip` command
__No arguments__
> /skip

## The `loop` command
`mode` Off
> /loop mode: Off

`mode` Queue
> /loop mode: Queue 

`mode` Song
> /Song mode: Song

## The `queue` command
__No arguments__
> /queue

## The `current` command
__No arguments__
> /current
